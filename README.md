# **好队友简介**

好队友是一款基于钉钉的数据处理及文件管理的工具、与阿里钉钉深度整合，  
帮助企业管理并挖掘数据和文档的价值。

#### 好队友云文件功能-让文件互联化



好队友的群文件的功能

* 自动同步钉钉组织构架，持续让组织在线；
* 实现多人多端在线实时编辑文档、表格、思维导图，让工作可以协同在线；
* 文件通过钉钉或钉消息快速分享，让文件信息的交互更加高效！

数据表的核心价值
